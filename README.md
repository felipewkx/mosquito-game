# Mosquito Hunter Game

Jogo arcade desenvolvido para navegador utilizando JavaScript puro, com foco em reflexos rápidos, interação dinâmica e progressão contínua de dificuldade. O objetivo é eliminar os mosquitos antes que o tempo acabe, enquanto a velocidade e a complexidade aumentam ao longo da partida.

---

## Demonstração

Acesse a aplicação: https://felipewkx.github.io/mosquito-game/

---

## Sobre o Projeto

O **Mosquito Hunter** foi desenvolvido com o objetivo de praticar conceitos fundamentais de desenvolvimento frontend utilizando tecnologias web nativas, sem dependência de frameworks externos.

O projeto explora lógica de programação, manipulação avançada do DOM, gerenciamento de estado e criação de mecânicas interativas em tempo real, proporcionando uma experiência inspirada em jogos arcade clássicos.

Além da jogabilidade dinâmica, o jogo conta com sistema de ranking local, efeitos sonoros, feedback visual e progressão automática de dificuldade.

---

## Funcionalidades

* Gameplay arcade em tempo real

## Mecânicas do Jogo

O nível de dificuldade aumenta automaticamente conforme o jogador avança na partida.

A cada cinco mosquitos eliminados, a velocidade de movimentação é intensificada, exigindo maior precisão, agilidade e tempo de resposta do jogador.

O sistema de ranking registra localmente as melhores pontuações utilizando o navegador, permitindo acompanhar a evolução do desempenho sem necessidade de backend.
